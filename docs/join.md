根据上面的规则，你可写出很多有意思的东西

如有标签的代码：

=== "C"

    ``` c
    #include <stdio.h>

    int main(void) {
      printf("Hello world!\n");
      return 0;
    }
    ```

=== "C++"

    ``` c++
    #include <iostream>

    int main(void) {
      std::cout << "Hello world!" << std::endl;
      return 0;
    }
    ```
=== "Go"

    ``` Go
    package main
    import "fmt"

    func main() {
        fmt.Println("Hello world!")
    }
    ```
